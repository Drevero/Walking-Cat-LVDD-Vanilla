# Vanilla code of the Walking cat from Les Vlogs d'un dev
Here is the repo of the walking cat from : https://lesvlogsdundev.com/

## Usage
Fork this repo or download it in .zip

**Make sure you put cat.css/cat.js and assets folder in your root project directory**

Next, you just have to open *index.html* and copy/paste the mandatory lines in your project.
